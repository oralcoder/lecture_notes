# 1. 강의 목표

- 웹 백엔드 아키텍처의 구성 원리를 이해하고 계층적 설계 역량 습득
- API 설계를 통해 실질적인 서비스 흐름 도식화구현 능력 함양
- 보안, 인증, DB 트랜잭션 등 백엔드 핵심 개념 습득
- Docker와 클라우드 환경을 활용하여 실제 서비스 수준의 배포/운영 능력 함양
